TP Exam 2025-2026 cours web

Dans ce TP, le but du jeu est de concevoir un site d-ecommerce pour une association sportive de Roundnet. Le site est composé de différentes routes :

  1. Une page produits qui affiche la liste des produits vendus
  2. Une page produit qui affiche le détails d'un produit
  3. Une page panier qui affiche le contenu du panier de l'utilisateur
  4. Une page info client qui affiche un formulaire pour récupérer les informations de l'utilisateur
  5. Une page résumé qui affiche le résumé de la commande
  6. Une page admin qui affiche la liste des commandes effectués

L'application est basée sur des données mockées qui sont stockées dans le dossier mocks. Elle n'est pas connectée à un back-end pour simplifier le développement.

Barème de notation à titre informatif (sur 15 points) :

Chaque page contient des consignes avec des actions à réaliser. La répartition des points est la suivante :

  1. Projet qui compile : 1 point
  2. L'application permet de créer une commande valide : 1 point
  3. Page produits : 2 points
  4. Page détails d'un produit : 2 points
  5. Page panier : 2 points
  6. Page info client : 5 points
  7. Page admin : 2 points
